# PaperHub 移动应用 - 中文使用指南

## 📱 应用介绍

**PaperHub** 是一个 React Native 移动应用，为您带来每日精选的学术论文。应用集成了三个主要研究方向的论文内容：

- **📹 Video Papers** - 视频相关的最新研究论文
- **🌍 World Model Papers** - 世界模型和环境建模论文
- **🤖 Agent Papers** - 智能体系统和自主代理论文

## 🚀 快速开始

### 安装应用

1. **从 GitHub Release 下载 APK**
   - 访问 [Daily Video Papers Release 页面](https://github.com/greasebig/daily-video-papers/releases)
   - 下载最新版本的 `PaperHub.apk` 文件

2. **安装到 Android 设备**
   - 在 Android 设备上启用"未知来源"应用安装
     - 设置 → 安全 → 允许安装未知来源的应用
   - 使用文件管理器打开下载的 APK 文件
   - 点击"安装"按钮

3. **首次启动**
   - 应用会自动加载最新的论文数据
   - 首次加载可能需要几秒钟

## 📖 使用说明

### 主屏幕
- 应用启动时显示主屏幕
- 点击顶部的导航按钮切换不同的论文分类
- 点击 GitHub 按钮可访问项目仓库

### 论文浏览
1. **选择分类** - 使用顶部导航按钮选择 Video、World Model 或 Agent
2. **查看论文列表** - 左侧显示可用的日期列表
3. **阅读论文** - 点击日期加载该日期的论文内容
4. **左下角导航按钮**
   - **← Back** - 返回上一篇已查看的论文
   - **Forward →** - 前进到下一篇已查看的论文
   - **⬆ Prev Day** - 查看前一天的论文
   - **⬇ Next Day** - 查看后一天的论文

### 高级功能
- **自动翻页** - 向下滚动到底部时自动加载前一天的论文
- **独立滚动** - 左侧日期列表和右侧论文内容可独立滚动
- **毛玻璃设计** - 现代化的界面设计，支持深色/浅色主题

## 🔄 更新应用

应用会定期从网站拉取最新的论文数据。如需手动刷新：
1. 返回主屏幕
2. 重新选择论文分类
3. 应用会自动获取最新数据

## 📲 系统要求

- **Android 版本** - Android 8.0 或更高版本
- **存储空间** - 至少 50MB 可用空间
- **网络连接** - 需要互联网连接以加载论文内容

## 🌐 上架到公共应用平台

### 方案 1：蒲公英 (Pgyer) - 推荐 ⭐

**优点**：完全免费、无需审核、支持中文、分享方便

**步骤**：
1. 访问 [蒲公英官网](https://www.pgyer.com/)
2. 注册账号（支持微信、QQ 登录）
3. 点击"上传应用" → 选择 APK 文件
4. 填写应用信息（名称、描述、版本号等）
5. 上传完成后获得下载二维码和短链接
6. 分享二维码或链接给用户

**分享方式**：
- 微信、QQ 群分享
- 社交媒体分享
- 论坛、社区发帖
- 个人网站或博客

### 方案 2：fir.im

**优点**：免费、支持版本管理、有统计数据

**步骤**：
1. 访问 [fir.im 官网](https://fir.im/)
2. 注册账号
3. 创建应用 → 上传 APK
4. 获得下载链接和二维码

### 方案 3：Google Play Store

**要求**：
- 需要 Google 开发者账号（一次性支付 $25）
- 需要通过 Google 的审核流程（通常 2-4 小时）
- 需要签名 APK（已在编译过程中完成）

**步骤**：
1. 创建 Google Play 开发者账号
2. 在 Google Play Console 中创建应用
3. 上传签名的 APK
4. 填写应用信息、截图、描述
5. 提交审核

### 方案 4：GitHub Release - 开发者友好

**优点**：与源代码仓库集成、版本管理清晰

**步骤**：
1. 在 GitHub 仓库中创建 Release
2. 上传 APK 文件到 Release
3. 添加更新日志和下载说明
4. 用户可直接从 GitHub 下载

## 📝 常见问题

**Q: 应用无法加载论文？**
A: 请检查网络连接，确保设备可以访问互联网。如问题持续，请清除应用缓存后重试。

**Q: 如何卸载应用？**
A: 长按应用图标 → 卸载，或在设置 → 应用管理中找到 PaperHub 并卸载。

**Q: 应用占用多少存储空间？**
A: 应用本身约 40-50 MB，论文数据会缓存在设备上。

**Q: 支持 iOS 吗？**
A: 当前版本仅支持 Android。iOS 版本正在开发中。
