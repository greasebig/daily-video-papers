# PaperHub APK Release

## 文件说明

- `PaperHub-v1.0.0-demo.apk` - 演示 APK（包含编译说明）
- `APK_BUILD_GUIDE.md` - 详细编译指南

## 安装步骤

1. 下载 APK 文件到 Android 设备
2. 启用"未知来源"应用安装
3. 打开文件管理器，找到 APK 文件
4. 点击安装

## 编译自己的 APK

请参考项目根目录的 `APK_BUILD_GUIDE.md` 文件。

## 系统要求

- Android 8.0 或更高版本
- 至少 50MB 可用存储空间
- 网络连接

## 功能特性

- ✅ 浏览每日学术论文
- ✅ 三个研究方向（Video、World Model、Agent）
- ✅ 左下角浮动导航
- ✅ 自动翻页功能
- ✅ 当前日期高亮显示
- ✅ 响应式布局
